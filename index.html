The Best Software for Laser Cutting

Instructions:

Right Arrow Key +1 width

Left Arrow Key -1 width

Up Arrow Key +1 height

Down Arrow Key -1 height

Q to +1 degree rotation

E to -1 degree rotation

W to +1 stroke weight

S to -1 stroke weight

Hold Shift to increase adjustments!

C for ellipse

R for rectangle

L for line

F for free draw (hold down the mouse in this mode)

T for text

G to toggle engraving

Z to undo last action (no redo available)
